# 项目更新操作记录

本文件用于记录这个网站项目的日常更新与发布流程，方便每次修改后快速同步到 GitHub 并触发 Vercel 自动部署。

## 日常更新流程（推荐）
```bash
1. 查看改动
git status

2. 添加改动
git add .

3. 提交改动
git commit -m "update content"

4. 推送到 GitHub（触发 Vercel 自动部署）
git push

```

```bash
cd "/Users/zsy/Desktop/zsy/zsy/个人网站"
git status

git push
```
